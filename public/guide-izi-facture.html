Le Guide d'Utilisation Complet (De A à Z)

Bienvenue dans le guide officiel d'utilisation de Izi-Facture. Cette application est conçue pour simplifier la gestion de vos clients, factures et finances. Suivez ce guide pour prendre en main toutes les fonctionnalités de l'outil.

💡 Astuce : Vous pouvez imprimer ce document (Ctrl+P ou Cmd+P) ou l'enregistrer au format PDF pour l'avoir toujours à portée de main !

1. Prise en main et "Mes Entreprises"

Izi-Facture est multi-entreprises. Vous pouvez gérer plusieurs entités (ou business) distinctes depuis le même compte.

Ajouter votre première entreprise

  1. Allez dans le menu latéral (à gauche).
  2. Cliquez sur le sélecteur d'entreprise tout en haut.
  3. Sélectionnez "Gérer mes entreprises".
  4. Renseignez le nom, l'email, l'adresse, choisissez une devise (ex: FCFA) et un logo (ex: 🏢).
  5. Cliquez sur "Ajouter cette entreprise". Elle apparaîtra instantanément.

2. Le Tableau de Bord (Dashboard)

Le tableau de bord est la première page que vous voyez. Il résume l'activité de l'entreprise actuellement sélectionnée.

  • Chiffre d'Affaires Global : La somme de toutes vos factures payées.
  • En Attente (Créances) : L'argent que vos clients vous doivent.
  • Filtre de Dates : Utilisez les champs "Du" et "Au" en haut pour filtrer les statistiques sur une période précise.
  • Graphique des Revenus : Affiche l'évolution de vos entrées d'argent par mois.

3. Gestion des Clients

Avant de créer une facture, il est recommandé de créer des clients dans votre carnet d'adresses.

Ajouter un client

  1. Cliquez sur Clients dans le menu latéral.
  2. Cliquez sur le bouton bleu "+ Nouveau Client" en haut à droite.
  3. Remplissez les informations (Nom, Contact, Email, Adresse).
  4. Cliquez sur "Créer le Client".

4. Création et Gestion des Factures

La fonctionnalité centrale d'Izi-Facture !

Créer une nouvelle facture

  1. Allez dans l'onglet Factures et cliquez sur "+ Nouvelle Facture".
  2. Sélectionnez un client existant (ou entrez son nom manuellement).
  3. Dates : Définissez la date d'émission et la date d'échéance.
  4. Articles : Ajoutez vos produits ou services. Renseignez la quantité et le prix unitaire, le total se calcule automatiquement. Vous pouvez appliquer une remise si besoin.
  5. Vérifiez la prévisualisation A4 en temps réel sur la droite (sur ordinateur).
  6. Cliquez sur "Enregistrer" ou "Enregistrer & Envoyer".

Statuts d'une Facture

  • Brouillon : Facture créée mais pas encore validée.
  • Envoyée : Transmise au client, en attente de paiement.
  • Payée : Montant réglé. L'argent est ajouté à vos revenus.
  • En retard : Date d'échéance dépassée, le client n'a toujours pas payé.
  • Annulée : Facture invalidée.

5. Envoyer une Facture (WhatsApp / Email)

Vous pouvez partager vos factures facilement depuis la vue détaillée d'une facture.

  1. Ouvrez une facture.
  2. Cliquez sur le bouton "Envoyer" en haut à droite.
  3. Choisissez le canal : WhatsApp (envoie un message pré-rempli) ou Email (envoie un brouillon via votre client mail par défaut).

6. Le Portefeuille (Wallet) et les Paiements

Izi-Facture intègre un suivi de votre trésorerie (Orange Money, MTN, Virement, Espèces).

  • Dépôt de fonds : Utilisez le bouton "Ajouter des fonds" (Flèche verte) pour enregistrer une rentrée d'argent manuelle.
  • Retrait : Utilisez le bouton "Retirer" (Flèche rouge) pour enregistrer une sortie ou dépense.
  • Clés d'API (Optionnel) : Vous pouvez renseigner des clés API dans la section Configuration Mobile Money si votre entreprise dispose d'une passerelle automatisée.

7. Rapports & Budgets

  • Budgets : Définissez des objectifs de revenus mensuels et suivez votre progression visuellement.
  • Rapports : Accédez à des analyses approfondies sur vos meilleurs clients, vos produits phares et l'évolution de vos finances sur l'année.

8. Paramètres et Personnalisation

Dans l'onglet Paramètres, vous pouvez :

  • Modifier votre profil utilisateur.
  • Changer le thème de l'application (Clair / Sombre / Système).
  • Gérer les notifications et les options de sécurité (Mot de passe).
